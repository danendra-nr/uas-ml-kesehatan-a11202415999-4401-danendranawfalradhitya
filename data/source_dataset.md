# Source Dataset Information

Dokumen ini menjelaskan asal-usul, kepemilikan, dan lisensi dari dataset yang digunakan dalam proyek klasifikasi tingkat risiko kesehatan ibu hamil (*Maternal Health Risk*).

## Informasi Sumber Dataset

*   **Nama Dataset:** Maternal Health Risk Data Set
*   **Sumber Portal:** UCI Machine Learning Repository
*   **Tautan Unduhan Resmi:** [UCI Maternal Health Risk Dataset](https://archive.ics.uci.edu/dataset/863/maternal+health+risk)
*   **Pemilik & Pengumpul Data Asli:** Marzia Ahmed, Laboratorium Biologi Komputasi & Bioinformatika, Universitas Swasta Bangladesh.
*   **Tahun Publikasi Dataset:** 2020

---

## Lisensi dan Penggunaan

*   **Lisensi:** Creative Commons Attribution 4.0 International (CC BY 4.0).
*   **Hak Cipta:** Bebas digunakan untuk keperluan riset akademis, pembelajaran, pengembangan model non-komersial, dan rekayasa ulang dengan syarat memberikan atribusi yang sesuai kepada penulis asli.
*   **Sitasi Akademik Resmi:**
    > Ahmed, M. (2020). *Maternal Health Risk [Dataset]*. UCI Machine Learning Repository. https://doi.org/10.24432/C5DP5D

---

## Batasan Etika & Privasi Data Pasien

*   **Masking Identitas:** Dataset ini telah melalui proses anonimisasi secara ketat. Tidak mengandung data pribadi sensitif pasien seperti nama, nomor telepon, alamat rumah, nomor identitas kependudukan (NIK), atau nomor rekam medis.
*   **Penggunaan Klinis:** Model machine learning yang dilatih menggunakan data ini dikembangkan untuk tujuan akademis dan skrining pendukung awal (*Decision Support System*). Model ini **tidak boleh** digunakan sebagai instrumen diagnosis pengganti dokter spesialis kandungan atau bidan berlisensi resmi.
